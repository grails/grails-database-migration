dbm-release-locks

Purpose

Releases all locks on the database changelog.

Description

Usage: 

grails [environment] dbm-release-locks --defaultSchema=[defaultSchema]

Required arguments: none .

Optional arguments:

  • defaultSchema - The default schema name to use

Note that the defaultSchema parameter name and value must be quoted if executed in Windows, e.g. 
grails dbm-release-locks "--defaultSchema=[defaultSchema]"
